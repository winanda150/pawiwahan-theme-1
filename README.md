# 💍 Pawiwahan Theme 1 – Digital Wedding Invitation

Pawiwahan adalah tradisi pernikahan dalam budaya Bali yang memiliki makna sakral serta penuh nilai adat dan budaya.

Website ini dirancang sebagai undangan pernikahan digital berbasis web (tema ke-1) dengan desain modern, elegan, dan responsif. Project ini menggabungkan nuansa budaya Bali dengan tampilan website yang modern, minimalis, dan nyaman digunakan di berbagai perangkat.

---

## 💻 Teknologi yang Digunakan

Website ini dibangun menggunakan:

- HTML → struktur halaman  
- CSS → desain dan tampilan visual  
- JavaScript → interaksi pengguna  

---

## ✨ Fitur Utama

- 💌 Tampilan undangan digital modern  
- 📱 Responsive design (desktop, tablet, dan mobile)  
- 🎨 Desain elegan dengan nuansa budaya Bali  
- 📍 Informasi acara pernikahan  
- 🖼️ Gallery foto pasangan  
- 🕒 Informasi waktu acara  
- 🎵 Background music untuk meningkatkan suasana  
- 💬 Kolom komentar ucapan dan doa dari tamu undangan  
- 👰 Mode khusus mempelai untuk pengelolaan tampilan undangan
- ✨ Animasi dan transisi antarmuka yang halus  

---

## 📦 Tujuan Pengembangan

Project ini dibuat sebagai:

- Media undangan pernikahan digital  
- Contoh tema website pawiwahan Bali  
- Latihan pengembangan frontend website
